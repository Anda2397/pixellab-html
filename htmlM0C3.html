Formulare HTML

Forumarele trebuie sa includa elemente de tip input, asa-numitele campuri.

Elemente principale

  • Input - elementul care permite userului sa interactioneze cu formularul, sa il completeze.
  • Button - elementul care permite userului sa anunte formularul ca si-a incheiat modificarile.

Form

Este elementul care inglobeaza campurile. Acest element poate primi un "method" care ne specifica felul de a trimite datele inapoi inspre server. Optiunile de retinut sunt "get" si "post".

Input

Atribute Input:
  • Placeholder - Un text ajutator care-i spune utilizatorului ce reprezinta campul respectiv. Acesta dispare odata ce utilizatorul incepe completarea inputului.
  • Type - Un atribut care are cateva valori fixe: text, password, email, tel, number (+others)
  • Required - Un atribut care poate fi scris sub forma required sau required=true care va marca un camp ca fiind obligatoriu
  • Minlength & maxlength - Un atribut numeric avand rolul de a specifica o lungime minima sau maxima pentru valoarea unui camp.
  • Value - Atributul care specifica valoarea predefinita a unui camp.
  • disabled - Atributul care ne permite dezactivarea unui camp
